📋 데이터

컬럼: user_id, session_id, event_type(view/add_to_cart/purchase), timestamp, device_type, region

규모: 100K+ 사용자, 6개월 데이터

SQL: 단계별 이탈율 계산 (view → add_to_cart → purchase)

Python: 시계열 트렌드 분석, 장치별/지역별 세그먼트 분석

  • 신규 고객 신뢰도 문제: 첫 구매 고객 이탈율 72%
  • 장바구니 이탈이 가장 큰 손실: 전체 이탈의 60%는 장바구니 단계에서 발생

전환율 개선: 2.8% → 3.5% (25% 상승)

추가 매출: 월 약 500만원 추가 수익 창출

고객 만족도: 결제 경험 NPS 점수 +8점 개선

  1. 모바일 UX 개선 - 결제 프로세스 단순화 (3단계 → 2단계)
  2. 장바구니 이탈 방지 - 24시간 이메일 리타겟팅 캠페인
  3. 신규 고객 유인 - 첫 구매 10% 할인 쿠폰

📍 기술: SQL, Python (pandas, matplotlib), Tableau

Subscription Churn Analysis

Python, SQL, Cohort

어떤 유저가, 왜 이탈하는가?

Cohort 분석을 통해 이탈 패턴을 추적하고, 고위험 유저 세그먼트를 식별하여 retention 전략을 수립했습니다.

Churn Rate ↓ 12%
LTV Impact +$2.5M

1단계: 코호트 정의

가입 월별 유저 그룹 생성, 월별 생존율 추적

2단계: 이탈 패턴 분석

고위험 코호트 식별 (가입 후 3개월 내 이탈율 60%)

3단계: 세그먼트 분석

유료 → 무료 다운그레이드 vs 완전 이탈 패턴 구분

  • 초기 30일이 critical period (60% 이탈 발생)
  • 프리미엄 기능 미사용 유저의 이탈율 85%
  • 고객 지원 연락 후 생존율 28% 증가
  1. 온보딩 프로그램 강화 (가입 후 7일 내 핵심 기능 경험)
  2. 고위험 세그먼트 조기 개입 (24시간 내 타겟팅)
  3. 재활성화 캠페인 (3개월 미사용 유저 대상)

예상 효과: 전체 churn rate 12% 감소

Sales Pattern Analysis

SQL, Excel, Tableau

언제·어디서 매출이 발생하는가?

시공간 데이터를 활용한 매출 패턴 분석으로 최적 재고 배치 및 마케팅 타이밍 전략을 제시했습니다.

지역 15 regions
매출 증가 +18%

1단계: 시계열 분석

월/주/시간대별 매출 트렌드 분석 (계절성 + 요일 효과)

2단계: 공간 분석

지역별(15개) 매출 분포, 성장률, 포텐셜 평가

3단계: 크로스 분석

지역 × 시간대 매트릭스로 최적 전략 도출

  • 금요일 오후 6-8시 피크 (주간 매출의 35% 집중)
  • 수도권 vs 지방 매출 비율 7:3 (개선 가능성 높음)
  • 월초/월말 수요 급증 (급여일 효과 명확)
  1. 피크타임(금토 저녁) 재고 집중 배치
  2. 지방 지역 마케팅 강화 (수도권 대비 ROI 20% 높음)
  3. 월초 프로모션 캠페인 확대

예상 효과: 매출 18% 증가 + 운영 효율 개선
