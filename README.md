# 📊 Portfolio example

안녕하세요.
저는 **데이터로 현상을 설명하고, 의사결정을 앞당기는 데이터 분석가**를 지향합니다.

이 레포지토리는 Python, SQL, Excel, Tableau를 활용해
**실제 회사에서 발생할 법한 비즈니스 문제를 어떻게 정의하고,
어떤 분석을 거쳐, 어떤 액션으로 연결했는지**를 정리한 포트폴리오입니다.

단순히 지표를 계산하는 데서 멈추지 않고,
"그래서 무엇을 바꿔야 하는가?"에 답하려는 관점으로 프로젝트를 구성했습니다.

---

## 🔍 What You'll Find Here

* **현실적인 비즈니스 문제 설정**

  * 구매 전환율 하락, 구독 이탈, 지역·시간대별 매출 차이 등
  * Kaggle 스타일의 추상적인 주제가 아닌, 실무 상황 기반 문제

* **End-to-End 분석 흐름**

  * 문제 정의 → 데이터 추출 → 분석 → 인사이트 → 액션 제안
  * 분석 결과가 실제 의사결정에 어떻게 쓰일 수 있는지까지 포함

* **도구를 "왜" 쓰는지에 대한 설명**

  * SQL: 지표의 기준을 만드는 도구
  * Python: 탐색·검증·자동화를 위한 도구
  * Excel: 빠른 검증과 커뮤니케이션
  * Tableau: 의사결정자를 위한 시각화

## 🎯 How I Work as an Analyst

저는 분석가를
**"숫자를 잘 보는 사람"이 아니라
"불확실한 상황에서 판단 기준을 만드는 사람"**이라고 생각합니다.

* 왜 이 지표를 봐야 하는지 설명할 수 있는가
* 이 결과가 어떤 결정을 바꾸는가
* 다음 액션은 무엇인가

이 질문에 답할 수 있는 분석을 목표로 합니다.
